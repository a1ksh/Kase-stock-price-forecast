# Kase-stock-price-forecast
This project is designed to analyze the shares of companies on the Kazakhstan Stock Exchange (KASE) and predict the price trend using the method of linear regression.

The project provides for shares of the following companies:
HSBK(Halyk Bank) 
KZTO(KazTransOil)
KZTK(Kazakhtelecom)
KEGC(KEGOC)
KZAP(Kazatomprom)
